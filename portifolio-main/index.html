Durante meus estudos, participei de diversos projetos que envolvem desenvolvimento web, banco de dados e criação de APIs. Nas horas vagas, adoro aprender novas linguagens e tecnologias.

Projeto cadastrado

Este projeto envolve uma plataforma de música para usuários cadastrados. Ele permite que os usuários criem uma conta, façam login e acessem uma seleção de músicas personalizadas. O foco está em fornecer uma experiência musical personalizada, com playlists e recomendações baseadas nas preferências dos usuários.

Flappy Bird

Um remake do famoso jogo Flappy Bird, onde o jogador precisa guiar um pássaro evitando obstáculos. Este projeto inclui a possibilidade de fazer escolhas diferentes em situações inesperadas, adicionando uma nova camada de desafio ao jogo original.

Projeto Gráfico

Um projeto interativo que utiliza comandos de voz para adivinhar um número secreto. O jogador tem a chance de interagir com o sistema por meio de reconhecimento de fala, tornando a experiência divertida e envolvente.

Vitoria Modas

Este é um remake do Flappy Bird, onde o pássaro foi substituído por um personagem personalizado feito por um aluno. Além do desafio de voar através dos obstáculos, o jogo ganha um toque pessoal e criativo.

Numero Mágio

Um projeto simples em Python que foca no uso de números mágicos. Ele é voltado para o entendimento e interação com medicamentos, onde o usuário pode explorar e aprender mais sobre medicamentos de maneira interativa.
